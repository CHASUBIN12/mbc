Emmet 기호 정리

Emmet에서 사용하는 주요 기호와 그 기능을 정리한 표입니다.

기호 설명 예시 결과
! HTML 기본 구조 생성 ! HTML5 기본 템플릿
> 자식 요소 생성 div>p <div><p></p></div>
+ 형제 요소 생성 h1+p <h1></h1><p></p>
^ 부모 요소로 이동 div>p>span^a <div><p><span></span></p><a></a></div>
* 반복 생성 ul>li*3 <ul><li></li><li></li><li></li></ul>
# id 속성 생성 div#logo <div id="logo"></div>
. class 속성 생성 div.container <div class="container"></div>
[] 속성 생성 input[type="text"] <input type="text">
{} 텍스트 내용 생성 a{Click me} <a>Click me</a>
